# Comparative-analysis-of-SRP
Soft Rot Pectobacteriacea is a disease caused by two different genus of pathogenic bacteria; Pectobacterium and Dickeya. And this project is all about comparing difference in the genetic constitution and functionality of various strains of these genus making them either competitive or coopertive with each other.
